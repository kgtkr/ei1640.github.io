前置き

ここまでで、入出力の仕方や、変数の扱いなど、プログラムを書く上での基本的な部分を学習してきたね。
今回は、プログラムの重要な動作の1つについて、学習していくよ。

基本的な構造

今更だけれど、プログラムを書く基本について、少し触れてみるよ。
過去にダイクストラという人が、「構造化プログラミング」というものを提唱したんだ。
全部理解しようとすると長くなってしまうから、興味があったら調べてみてね。

構造化プログラミングでは、3つの制御構造しか使わないんだ。
「順次構造」「選択構造」「反復構造」の3つだよ。

・順次構造・
言葉の意味通り、処理が順番通り、1つずつこなされていくものだよ。
今まで書いてきたプログラムは、全部順次構造だね。

・選択構造・
今回学習するものだよ。
「もしも〜だったら、こうする。」という感じの動作だよ。

・反復構造・
別のところで学習するものだよ。
「〜の間、こうする。」という感じの動作だよ。

「プログラミング」って聞くと難しそうだったかもしれないけれど、3つの制御構造しか使わないって言われると、できる気がしてくるね!!

if文

長くなってしまったけれど、先ほど言った通り、今回は選択構造をやるよ。
C言語においては、if文というものが選択構造に当たるよ。

早速使い方をやっていこう。 

#include <stdio.h>
int main(){

  if(/* 条件式 */){
    // 処理
  }

  return(0);
}

使い方は上の通りだけど、このままでは動かないよ。
「条件式」がないと動かないんだ。ここは処理によって色々と変わるよ。

条件式というのは、それが正しかったら「真」、正しくなかったら「偽」が返ってくるものだよ。
「返ってくる」というと分かりづらいから、今は「真または偽になるもの」って思っておいてね。
条件式には色々あるよ。それについては、下の方でまとめてあるからみてみてね。

if文において、条件式が真だった場合、上記のソースの「// 処理」の部分が実行されるんだ。
偽だった場合は、「// 処理」の部分は実行されないで、無視されるよ。


条件式に当てはまった場合は処理Aをして、当てはまらなかった場合は処理Bをする、なんてこともできるよ。 

#include <stdio.h>
int main(){
  
  if(/* 条件式 */){
    // 処理A
  } else {
    // 処理B
  } 

  return(0);
}

このように、「else」を使うことによって、条件式に当てはまった場合、当てはまらなかった場合のように処理を分けることができるよ。
また、こんな書き方もできるよ。 

#include <stdio.h>
int main(){
  
  if(/* 条件式 */){
    // 処理
  } else if(/* 条件式 */){
    //処理
  }

  return(0);
}

「else」と「if」をつなげることもできるんだね!!
これも、競プロで使うことがあるから、覚えておこう。

条件式

ここまでで、「if文」というものはわかったけど、「条件式」が分からなかったから、実際に動かしてみることができなかったね。
条件式を学習して、「if文」を使ってみよう!!

条件式は色々あるから、よく使うものだけまとめてみたよ。
直感的に分かるものから、プログラミング特有のものまであるけれど、最低限ここにあるものは覚えておくといいよ。
条件式を考えるのが苦手な人もいるかもしれないけれど、ゆっくり慣れていけばいいから大丈夫!!
if文が使えるだけで、解ける問題の幅が広がるから、頑張ってみよう。

演算子
意味
<
 数学と同じように、大小の比較で使うよ。
右が左より大きかったら、真を返すよ。
>
 数学と同じように、大小の比較で使うよ。
右が左より小さかったら、真を返すよ。
<=
 これも、大小の比較で使うものだよ。
右が左以上だったら、真を返すよ。その数を含むか含まないかの違いだね。
>=
 これも、大小の比較で使うものだよ。
右が左以下だったら、真を返すよ。その数を含むか含まないかの違いだね。
==
 右と左が等しかったら、真を返すよ。
!=
 右と左が違ったら、真を返すよ。

条件式を使ってみると、こんな感じのソースが書けるようになるね!! 

#include <stdio.h>
int main(){
  // 変数の宣言
  int a, b;
  
  // 変数の入力
  scanf("%d %d", &a, &b);

  // 比較
  if(a > b){
    // aがbより大きい
    printf("最初に入力された数が大きいです。\n");
  } else if(a < b){
    // aがbより小さい
    printf("二番目に入力された数が大きいです。\n");
  } else if(a == b){
    // aとbが等しい
    printf("二つの数は等しいです。\n");
  }
  
  return(0);
}
