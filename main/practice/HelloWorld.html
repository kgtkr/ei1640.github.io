プログラミングを始めよう!!

「プログラミング」って、どんなイメージがあるかな？
難しいイメージを持ってる人もいるかもしれないし、簡単だと思っている人もいるかもしれないね。

ここでは、「競技プログラミング」を始めるための、プログラミングの基本を、Hamako Online Judge(通称HOJ)と、C言語やC++を使って学んでいくよ!!

まず初めに、「Hello World!」って出力(表示)してみよう!!

#include <stdio.h>
int main(){
  printf("Hello World!\n");    
  return(0);
}

出力できたかな?

どうして出力できたのか、予想してみよう。ソース内で「 Hello World!」の記述があるのは、3行目だけだね。「 printf」を使うと、「 Hello World!」って出力ができるみたい。

確認のために、「 Hello World!」の文字を変えて実行してみよう!!
予想通りの出力になったかな?

次に、「 Hello World!」の後にある、「 \n」の意味を考えてみよう。
これは「改行」を意味するものだよ。 競プロでは毎回使うので、覚えていこう。
「 printf」は、「 "」の間にある文字を出力してくれる「関数」なんだ。関数については、別のところで詳しくやるから、今はそういう名前なんだって覚えておいてね。

「 printf」や「 return(0)」の後にある「 ;」はなんだろう?
これは、日本語でいうと「。」のような役割で、「文章の終わりにつけるもの」みたいな感じだよ。
今後、色々な命令を学習する時にも、最後に「 ;」がついていることが多いから、覚えておこうね。

では、3行目以外のプログラムについて考えてみよう…って言いたいところだけど、まだ難しいので、「プログラムを書くときのおまじない」って思っててね。

ここまでできたら、HOJの 「001 Hello World!」が解けるよ!

コメント

最後に１つ、ソースの中に、「コメント」を記述できる、ってことは覚えておこう。
「 //」って書くと、その行の「 //」以降の文字は全部コメントになるよ。
「 /*」と「 */」で囲むと、囲んだ範囲だけ、コメントになるよ。

コメントは、ソースの中に記述しても動作には影響を及ぼさないので、積極的に利用していこう。特に、長いプログラムを書くときは、どこで何をしているか、ソースを読まなくてもわかるように、コメントで記述しておくといいね!!

#include <stdio.h>
int main(){
  printf("Hello World!\n"); // これはコメントです
  /* これもコメントです */

  /*
    こんな風にも使えるよ
  */
  return(0);
}
