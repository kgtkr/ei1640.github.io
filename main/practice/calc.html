演算

前回、変数の作り方と出力の仕方を勉強したね!!
今回は、演算のことを学習するよ!!

変数は、数字や文字を入れる「箱」って説明したね。
数学では、その変数を使って、足し算したり、引き算したり…ってしたことがあると思う。
プログラミングでも、同じように、足し算したり、引き算したり…することができるんだ。 

#include <stdio.h>
int main(){
  // 変数の宣言
  int a = 10;
  int b;

  // 足し算
  b = a + 10;
  printf("%d\n", b);

  // 引き算
  b = 100 - a;
  printf("%d\n", b);

  // 掛け算
  b = a * a;
  printf("%d\n", b);

  // 割り算
  b = 100 / 10;
  printf("%d\n", b);

  // あまりを求める
  b = 123 % 100;
  printf("%d\n", b)
  
  return(0);
}

ソースコードにコメントで書いてある通り、四則演算とあまりを求めることができるよ!!
掛け算と割り算だけ、「 *」と「 /」を使うから、気をつけてね。

「 %」であまりを求めることができるのは、プログラミング特有だね。
最初のうちはあまり使わないかもしれないけれど、少し難しい問題になるとよく使うから覚えておこうね!! 

#include <stdio.h>
int main(){
  // 変数の宣言
  int a,b;
  
  // 数字同士を足すこともできるよ
  a = 1 + 2;
  printf("%d\n", a);

  // 計算式を、printfにそのまま書き込むこともできるよ 
  printf("%d\n", 10/2);

  // 四則演算を混ぜて行うこともできるよ
  // 掛け算と割り算が優先されるよ
  b = 3 * 6 + 9 / 3;
  printf("%d\n", b);

  // ()をつけると、()の中が優先的に処理されるよ
  b = 3 * (6 + 9) / 3;
  printf("%d\n", b);

  // こんなこともできるよ
  // 数学的にはおかしいけれど、プログラミングの世界では正しいよ
  // aの値に1を足した数がaの中に上書きされるって感じ aの中身は11になるんだ
  a = 10;
  a = a + 1;
  
  return(0);
}

プログラム特有の書き方もあるよ。 

#include <stdio.h>
int main(){
  // 変数の宣言
  int a;

  // 値の代入
  a = 10;

  /** こんな書き方もできるよ **/
  
  // += -= *= /= %= って書くことができるよ
  // a = a + 10 という意味になるよ
  a += 10;
  // a = a % 10 という意味になるよ
  a %= 10;

  // インクリメント
  // a = a + 1 と同じ意味だよ
  a++;

  // デクリメント
  // a = a - 1 と同じ意味だよ
  a--;

  return(0);
}

割り算について

プログラミングにおいての割り算は、型によって少し変わるんだ。
例えば、「整数」を扱う型を 使って、割り切ることのできない演算をすると、どうなるだろうか。 

#include <stdio.h>
int main(){
  // 変数の宣言
  int a;

  // 割り切れる割り算
  a = 9 / 3;
  printf("%d\n", a);

  // 割り切れない割り算
  a = 10 / 3;
  printf("%d\n", a);

  return(0);
}

割り切れるものは正しく表示できたが、割り切れないものは、あまりを無視して整数で表現できる数が表示されたと思う。
このように、整数での割り算は、正しい値にはならないので、気をつけていこう。

実数を扱うことのできる型を使えば、小数点以下の数も表示されるよ。 

#include <stdio.h>
int main(){
  // 変数の宣言
  double a, b, c;

  // 値の代入
  a = 10;
  b = 3;
  
  c = a / b;
  
  printf("%lf\n", c);
  
  return(0);   
}

キャスト

int型の変数とdouble型の変数で、演算はできると思う？
答えは、「できるけど、そのままではできない」って感じかな。
「キャスト」というものを使えば、違う型の変数同士で演算することができるよ!! 

#include <stdio.h>
int main(){
  // 変数の宣言
  int a;
  double b, c;

  // 値の代入
  a = 10;
  b = 10;

  // 演算
  c = b + (double)a;

  // 出力
  printf("%d\n", a);

  return(0);
}

12行目で使われているものが「キャスト」だよ。
()の中に、どの型に変換したいかを記述してあげるだけで、その後ろにある変数、ソースコードの中で言えば「 a」の型が変わるよ。
